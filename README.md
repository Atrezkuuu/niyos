# NIYOS
NIY это минималистичная x16 ОС на fasm, которую я планирую обновлять.
## Запуск
Для работы ОС нужен искуственный hdd.img qemu команда для создания которого есть в Makefile
```
make createhdd
```
После сборки и создания hdd.img вы можете воспользоваться командами Makefile для запуска с разными устройствами
```
make load <- Загрузка с двумя дисками, приоритет niyos.bin
make loadhdd <- Загрузка только hdd.img
```
## Тесты и предостережение
Я протестировал ОС только на qemu и результат запуска на реальном оборудовании может быть ФАТАЛЬНЫМ!
